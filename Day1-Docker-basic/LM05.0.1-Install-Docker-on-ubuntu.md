## ✅ Clean Docker installation on Ubuntu 20.04 (focal)

### 1) Cleanup old/broken Docker repo + packages

```bash
sudo apt-get remove -y docker docker-engine docker.io containerd runc docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
sudo rm -f /etc/apt/sources.list.d/docker.list
sudo rm -f /etc/apt/keyrings/docker.gpg
sudo apt-get update
```

### 2) Install prerequisites

```bash
sudo apt-get install -y ca-certificates curl gnupg
```

### 3) Add Docker GPG key (keyrings method)

```bash
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg
```

### 4) Add Docker repo for **focal**

```bash
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu focal stable" | \
sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
```

### 5) Update & install Docker

```bash
sudo apt-get update
sudo apt-get install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin
```

### 6) Enable + start Docker

```bash
sudo systemctl enable --now docker
```

### 7) Allow running docker without sudo

```bash
sudo usermod -aG docker $USER
newgrp docker
```

### 8) Test

```bash
docker run hello-world
docker version
docker compose version
```
